# PathoLogic AI - 设计实现指南

## 概述

本指南为 PathoLogic AI 系统的所有功能页面提供统一的设计规范和实现方案。

---

## 1. 页面设计规范

### 1.1 AI 阅片室 (SlideAnalyzer)

**设计特征**：
- 主色调：蓝色系 (#3b82f6)
- 布局：左侧控制面板 (w-80) + 右侧预览区
- 关键组件：上传区、分析按钮、结果展示

**页面结构**：
```
┌─────────────────────────────────────────┐
│ 页面标题 (蓝色图标)                      │
├──────────────┬──────────────────────────┤
│ 上传区       │ 预览/结果区              │
│ 分析按钮     │ (flex-1)                 │
│ 参数设置     │                          │
│              │                          │
└──────────────┴──────────────────────────┘
```

**实现要点**：
- 使用 `SidebarLayout` 组件
- 上传区使用虚线边框 + 拖拽提示
- 分析按钮：主色蓝色，加载时显示旋转图标
- 结果展示：卡片网格布局，支持滚动

- 主色调：青色系 (#06b6d4)
- 布局：左侧预设列表 + 右侧 3D 模型
- 关键组件：预设按钮、自定义输入、模型查看器

**页面结构**：
```
┌─────────────────────────────────────────┐
│ 页面标题 (青色图标)                      │
├──────────────┬──────────────────────────┤
│ 预设模型列表 │ 3D 模型显示区            │
│ 自定义输入   │ (flex-1)                 │
│ 生成按钮     │                          │
│              │                          │
└──────────────┴──────────────────────────┘
```

**实现要点**：
- 预设按钮：hover 时显示 shadow-lg

## 4. 交互规范

### 4.1 悬停效果

- **卡片**：shadow-sm → shadow-lg + border-slate-300
- **按钮**：背景色变深 + 文字颜色变化
- **链接**：颜色变深 + 下划线

### 4.2 加载状态

- **按钮**：disabled + 旋转图标 + 文字变化
- **页面**：LoadingState 组件 + 旋转动画
- **列表**：骨架屏或加载指示器

### 4.3 空状态

- **列表**：EmptyState 组件 + 图标 + 文字 + 操作按钮
- **搜索**：无结果提示 + 清除搜索建议

### 4.4 错误状态

- **表单**：红色边框 + 错误提示文字
- **页面**：Alert 组件 + 重试按钮
- **操作**：Toast 通知或模态框

## 5. 响应式设计

### 5.1 断点

```
sm: 640px   - 手机横屏
md: 768px   - 平板
lg: 1024px  - 桌面
xl: 1280px  - 大屏
```

### 5.2 侧边栏

- **lg 以上**：显示固定侧边栏
- **md 以下**：隐藏侧边栏，使用汉堡菜单

### 5.3 网格

- **mobile**：1 列
- **sm**：2 列
- **md**：2 列
- **lg**：3-4 列

## 6. 实现检查清单

### 视觉质量
- [ ] 所有图标来自 Lucide React
- [ ] 无 emoji 图标
- [ ] 悬停状态不导致布局抖动
- [ ] 颜色对比度满足 WCAG AA

### 交互
- [ ] 所有可点击元素有 cursor-pointer
- [ ] 悬停状态清晰可见
- [ ] 过渡效果平滑 (150-300ms)
- [ ] 加载状态有反馈

### 响应式
- [ ] 在 375px 宽度下可用
- [ ] 在 768px 宽度下可用
- [ ] 在 1024px 宽度下可用
- [ ] 无水平滚动条

### 可访问性
- [ ] 所有图片有 alt 文本
- [ ] 表单输入有标签
- [ ] 颜色不是唯一指示器
- [ ] 支持键盘导航

## 9. 常见模式

### 9.1 左侧栏 + 右侧内容

```tsx
<SidebarLayout
  sidebar={<SidebarContent />}
>
  <PageHeader title="..." />
  <div className="flex-1 overflow-y-auto p-6">
    {/* 主内容 */}
  </div>
</SidebarLayout>
```

### 9.2 顶部操作栏 + 表格

```tsx
<div className="h-screen flex flex-col">
  <PageHeader title="..." />
  <div className="p-6 border-b border-slate-200 flex gap-4">
    {/* 搜索、筛选、操作按钮 */}
  </div>
  <div className="flex-1 overflow-y-auto">
    {/* 表格 */}
  </div>
</div>
```

### 9.3 网格卡片

```tsx
<div className="p-6">
  <Grid cols={3} gap="md">
    {items.map(item => (
      <Card key={item.id} hover>
        <CardBody>{item}</CardBody>
      </Card>
    ))}
  </Grid>
</div>
```
